# Superset (superset-sh)

> Category: Agent Orchestrator (Tier A — direct competitor)
> Website: https://superset.sh
> Type: Electron desktop
> Price: Free (ELv2)
> Stars: 2k
> OSS: Elastic License v2 (not truly OSS)
> Desktop: Yes (Electron)

## Summary
Closest architectural match to SlayZone — Electron + xterm.js + Monaco + SQLite + browser pane + worktrees. Supports 6+ providers. No kanban or task management (terminal multiplexer, not PM). Electric SQL for cloud sync. Very new (~4 months old).
