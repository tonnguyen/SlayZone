Frequently Asked Questions

Everything you need to know about the local-first AI coding orchestrator.

How does SlayZone handle AI agents?

SlayZone is a powerful orchestrator. It runs agents like Claude Code or Gemini CLI in real PTY sessions on your machine. You use your own agents and keysâ€”SlayZone provides the isolated environment and visual interface to manage them.

Is my code or data sent to your servers?

Never. SlayZone is fully local-first. All your code, task descriptions, terminal logs, and git worktrees stay exactly where they belong: on your machine. There is no cloud, no account, and no tracking.

Which agents are supported?

Any agent that runs in a terminal. We have built-in optimizations for Claude Code (including auto-status tracking), but SlayZone works seamlessly with any CLI tool you already use.

What is the "slay" CLI?

It's a command-line interface that ships with the app. It lets you interact with your SlayZone board directly from your shell. You can create tasks, mark them done, or list status without ever leaving your main terminal. The app live-updates as you type.
